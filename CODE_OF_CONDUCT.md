# 贡献者行为准则 · Contributor Covenant Code of Conduct

## 我们的承诺

我们作为成员、贡献者和维护者，承诺让每个人在参与我们社区时都**不受骚扰**，无关其年龄、体型、明显或不明显的残疾、种族、性征、性别认同和表达、经验水平、教育程度、社会经济状态、国籍、相貌、人种、宗教信仰或性取向。

我们承诺以积极贡献于开放、友好、多元、包容、健康社区的方式行事和互动。

## 我们的准则

有助于创造积极环境的行为包括：

- 对他人表现出**同理心**与**善意**
- 尊重**不同的意见、观点和经验**
- 提出并**优雅地接受**建设性批评
- 为自己的错误**承担责任**并向受影响方道歉，从中吸取经验
- 着眼于**整个社区的最大利益**，而不仅仅是个人

不可接受的行为包括：

- 使用性化的语言或图像，以及性关注或挑逗
- 嘲讽、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息（如住址、电子邮箱）
- 在专业场合被合理认定为不适当的其他行为

## 执行责任

社区维护者有权且有责任对任何他们认为不符合本行为准则的行为删除、编辑或拒绝评论、commit、代码、wiki 编辑、issue 及其他贡献，并在合适时**说明理由**。

## 适用范围

本行为准则适用于所有社区场合，也适用于当个人在公共场所代表社区时。代表社区的例子包括：使用官方邮箱、通过官方社交媒体账号发帖，或在线上线下活动中作为指定代表出席。

## 执行

辱骂、骚扰或其他不可接受的行为可通过以下方式向维护者报告：

- **GitHub Issue**（隐私敏感的不要走这个渠道）
- **维护者邮件**：通过维护者 GitHub profile（[@Lau0x](https://github.com/Lau0x)）公开的联系方式

所有投诉都会被**及时公平地审查和调查**。

维护者有义务为投诉者保密。具体的执行细则可能另行公布。

## 执行准则

维护者在判定本行为准则违规后果时，将遵循以下社区影响准则：

### 1. 更正

**社区影响**：使用了不恰当的语言或被社区视为不专业、不受欢迎的行为。

**后果**：维护者发出**私下书面警告**，说明违规原因及该行为为何不当。可能要求公开道歉。

### 2. 警告

**社区影响**：单次事件或一系列行为的违规。

**后果**：对**持续行为**的警告。在指定时间内不得与相关人员互动（包括主动与执行行为准则者互动），此限制适用于所有社区渠道与外部渠道（如社交媒体）。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区准则，包括持续不当行为。

**后果**：在指定时间内**临时封禁**与社区的任何互动或公开交流。此期间不允许以任何方式与相关人员互动，包括主动与执行行为准则者互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区准则的模式，包括持续不当行为、骚扰个人或对个人群体的攻击或贬损。

**后果**：永久禁止在社区内以任何形式公开互动。
